# 🍌 Banana Laundering Map

A geo-tasking application for tracking banana delivery locations with real-time mapping, built with Next.js 15, TypeScript, and Google Maps.

![Banana Laundering Map](https://img.shields.io/badge/Banana-Laundering-yellow) ![Next.js](https://img.shields.io/badge/Next.js-15-black) ![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)

## 🌟 Features

- **Interactive Google Maps** with custom markers
- **Real-time location tracking** with live user position
- **Task Management** - Create, read, update, and delete delivery tasks
- **Image Upload** - Attach photos to tasks with preview
- **Quantity Tracking** - Track banana quantities with visual indicators
- **Status Management** - Mark tasks as pending or delivered
- **Responsive Design** - Works on desktop and mobile
- **Modern UI** - Glass-morphism design with Tailwind CSS
